Sección de Timumachtikan

Yek peyna, yek teutak wan yek peyna, ken tinemi? yek? nusan yek!

Descripción

Este proyecto propone el desarrollo de una aplicación educativa para la enseñanza del Nawat (también llamado Náhuat o Pipil), una lengua indígena en riesgo de extinción. A través de una base de datos estructurada y métodos de aprendizaje, la aplicación facilita la enseñanza del idioma de manera intuitiva y culturalmente relevante.

Detalles

Este propotipo busca mejorar para revitalizar el idioma mediante tecnología educativa accesible y bases de datos estructuradas que permitan un aprendizaje interactivo desde niveles básicos hasta avanzados.
